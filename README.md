# Simple Java lotto number generation app

A basic Java application that emulates a lottery slikpik ticket

## Requirements
- Java Development Kit (JDK)
- Windows / macOS / Linux

## How to Run
1. Open your terminal or command prompt.
2. Navigate to the directory where the files were cloned.
Compile the Java file into bytecode using javac:
```bash
javac lotto.java
```

3. Run the compiled Java class using the java command
```bash
java lotto
```

Alternatively, install the 'Extension Pack for Java' in VS Code, then click the "Run Java" or "Debug Java" text that appears directly above the main method declaration
